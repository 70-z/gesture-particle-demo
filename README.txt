青年才俊

运行方式：
1. 双击 start-demo.bat。
2. Edge 打开后允许摄像头权限。
3. 也可以直接访问 GitHub Pages 公网页面。

普通模式：
- 手势 1 或按钮 1：粒子组成“青年才俊”。
- 手势 2 或按钮 2：粒子组成“苏音唯美，爱泷不悔”。
- 手势 3 或按钮 3：粒子组成“Silence牛逼克拉斯”。
- 手势 4 在普通模式下不显示文字。
- 文字周围会有不规则粉色粒子漂浮。
- 点击“相册”：进入相册文件夹界面。

相册模式：
- 默认照片保存在“20260509”文件夹。
- “cj”是相册里的一个文件夹。
- 点击文件夹标签可切换目录。
- 照片真实显示在中间，周围是粉色粒子特效。
- 页码如 1/4 由粉色粒子组成。
- 手势 1 / 2 / 3 / 4 或按钮 1 / 2 / 3 / 4：切换对应照片。
- 点击照片两侧的左右按钮：上一张 / 下一张。
- 点击“退出”：退出相册。

文件夹和上传：
- 点击“新建”可新建相册文件夹。
- 上传前在下拉框里选择目标文件夹。
- 点击“上传”可把本机照片加入选中的文件夹。
- 点击“删除”可删除当前文件夹正在显示的照片，并同步更新 GitHub 相册清单。
- 为保证多设备可见，新建文件夹和上传照片会写入 GitHub 仓库。
- 第一次上传前点击“授权”，填写 GitHub fine-grained token，权限需要 Contents: Read and write，范围选择 70-z/qingniancaijun 仓库。
- 授权只保存在当前浏览器本地，不会写进代码仓库。
- 上传成功后，其他设备刷新 GitHub Pages 公网页面即可看到。

如果摄像头无法打开：
- 关闭微信、会议软件、其他正在使用摄像头的网页。
- 在 Edge 地址栏左侧权限图标里允许摄像头。
- 回到页面后点击“摄像头”按钮重试。
