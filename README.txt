3D 手势粉色粒子系统

运行方式：
1. 双击 start-demo.bat。
2. Edge 打开后允许摄像头权限。
3. 也可以直接访问 GitHub Pages 公网页面。

普通模式：
- 手势 1 或按钮 1：粒子组成“青年才俊”。
- 手势 2 或按钮 2：粒子组成“才智超群”。
- 手势 3 或按钮 3：进入默认相册。

相册模式：
- 照片真实显示在中间，周围是粉色粒子特效。
- 页码如 1/4 由粉色粒子组成。
- 手势 1 / 2 / 3 / 4 或按钮 1 / 2 / 3 / 4：切换对应照片。
- 识别到两只手或点击“退出”：退出相册。

cj 相册：
- 点击 cj 打开 cj 相册。
- 点击“上传”可把本机照片加入 cj 相册。
- GitHub Pages 是静态网页，上传的 cj 照片会保存在当前浏览器本地，不会上传到服务器。

如果摄像头无法打开：
- 关闭微信、会议软件、其他正在使用摄像头的网页。
- 在 Edge 地址栏左侧权限图标里允许摄像头。
- 回到页面后点击“摄像头”按钮重试。
