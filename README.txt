3D 手势粒子系统

运行方式：
1. 解压整个文件夹。
2. 双击 start-demo.bat。
3. Edge 打开后允许摄像头权限。

功能：
- 手势 1：粒子组成“青年才俊”。
- 手势 2：粒子组成“长长久久”。
- 双手张合：控制粒子收缩与扩散。
- 键盘 1 / 2 可作为备用切换方式。

环境要求：
- Windows
- Microsoft Edge
- Python 3
- 首次运行需要联网加载 Three.js 和 MediaPipe 手势识别组件。

如果摄像头无法打开：
- 关闭微信、会议软件、其他正在用摄像头的网页。
- 在 Edge 地址栏左侧权限图标里允许摄像头。
- 回到页面后点击左下角“摄像头”按钮重试。
